# Card News - 여행 블로그 프로젝트

## 프로젝트 소개
Card News는 여행과 모험에 관한 이야기를 공유하는 현대적인 블로그 플랫폼입니다. Next.js와 TypeScript를 기반으로 구축되었으며, Tailwind CSS를 사용하여 반응형 디자인을 구현했습니다.

## 주요 기능
- 피처드 포스트 섹션
- 최근 게시물 표시
- 카테고리별 내비게이션
- 반응형 카드 레이아웃
- 다크모드 지원

## 기술 스택
- Next.js 15.0.3
- React 19.0.0
- TypeScript
- Tailwind CSS
- Geist 폰트 시스템
